# 🛒 E-Commerce Backend Application

## 📌 Overview

This is a backend-focused **E-Commerce Application** built using Spring Boot.
It includes secure authentication, role-based access control, payment integration, and cloud deployment.

---

## 🚀 Tech Stack

* **Backend:** Java, Spring Boot
* **Security:** Spring Security (JWT)
* **Database:** MySQL (JPA / Hibernate)
* **API Testing:** Postman, Swagger UI
* **Payment:** Stripe API
* **Deployment:** AWS (EC2, RDS, S3)

---

## 🔥 Features

* 🔐 JWT Authentication (Sign In / Sign Out)
* 👥 Role-Based Authorization
* 🛍️ Product & Order APIs
* 💳 Stripe Payment Integration
* 📊 Dashboard APIs
* 🔗 Entity Relationships (JPA/Hibernate)
* 📡 API Testing using Swagger & Postman
* ☁️ Deployed on AWS

---

## 🔐 Security Flow

```id="flow2"
Request → JWT Filter → Authentication → Authorization → Controller
```
